# RTTK-with-React
摘肺站是一個右肺摘除所翻譯的圖像小說的進度追蹤、資源歸檔網站。這個項目是使用React重構舊版摘肺站的嘗試。
榮光歸於地瓜蛋教皇，阿米阿米。
